Class: ItemsFieldMask

ItemsFieldMask()

An ItemsFieldMask keeps track of fields masked by category, as well as some globally masked Item fields (currently labels, uplinks and downlinks). This class is used by the user of the SDK to narrow the set of fields brought down in a search query, and then to allow the SDK to safely construct partial items from those results.
